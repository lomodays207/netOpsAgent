# 实现计划: RAG证据来源展示功能

## 概述

本功能在现有RAG检索基础上，扩展后端API以返回证据来源信息，并在前端展示这些证据，支持用户点击查看完整文档内容。实现包括：
- 后端SSE事件流扩展（新增evidence_sources事件）
- 新增文档查询REST API端点
- RAG检索层增强（返回文档ID和元数据）
- 前端证据来源面板和预览模态框
- 文档内容缓存和安全防护
